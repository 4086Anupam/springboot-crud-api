# springboot-crud-api
This project is a RESTful API built using Spring Boot that performs basic CRUD (Create, Read, Update, Delete) operations. It follows a clean layered architecture and demonstrates how to build scalable backend services.

📌 Features
Create new records
Retrieve all records
Retrieve record by ID
Update existing records
Delete records
Proper use of HTTP methods (GET, POST, PUT, DELETE)

🧩 Project Structure
Controller → Handles incoming HTTP requests
Service → Contains business logic
Repository → Interacts with database
Entity → Represents database table

🔗 API Endpoints
Method	Endpoint	Description
POST	/users	Create user
GET	/users	Get all users
GET	/users/{id}	Get user by ID
PUT	/users/{id}	Update user
DELETE	/users/{id}	Delete user

🎯 Learning Outcome
Understanding of REST API design
Implementation of layered architecture
Hands-on experience with Spring Boot and JPA
Handling HTTP requests and responses
